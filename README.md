# Node.js

##

## Project setup

```
npm install

npm install --save forecast

nodemon app.js -- --config # pass config to app.js

npm install nodemon@3.0.3 --save-dev

npm update

npm audit fix --force

npx kill-port --port 8080
```

### Run

```
node server.js

npm run start

npm run dev
```
